# A-knowledge-Discovery-Framework-for-High-Dimensional-Youth-Risk-Behavior
CDC YRBS data mining project analyzing adolescent risk behaviors (1991–2017) across 1.17M records. Covers exploratory data analysis, geographic mapping, confidence interval analysis, and predictive modeling using Random Forest, XGBoost, Gradient Boosting, and CatBoost. Python | Scikit-learn | SHAP
